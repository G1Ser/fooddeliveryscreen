# echart

这是一个基于vue3框架的项目。是一个新中地外卖平台大屏可视化项目，结合vue-echarts来绘制各种图表，通过这个项目了解echarts的使用，希望这个项目对你有所帮助！


## 项目配置
去高德地图申请一个Web服务密钥。
```sh
//index.html 8行 设置你Web服务的密钥
    <script type="text/javascript" src="https://webapi.amap.com/maps?v=2.0&key="<Your Key>"></script> 
```


### 项目运行

```sh
npm install

npm run dev
```
